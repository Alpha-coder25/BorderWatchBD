BorderWatch BD
Citizen-driven border safety and crime reporting platform for Bangladesh border areas.
Project Vision
BorderWatch BD allows residents, travellers, and local communities to report border-related
Incidents and find the nearest BGB camp for emergency support.
Core Objectives
1. Crime Data Collection- Push-in attempts- Border killings- Smuggling- Human trafficking- Missing persons- Suspicious activities
2. Emergency Support- Nearest BGB camp finder- One-click call- Route directions
3. Border Awareness- Safety guidelines- Emergency notices- Restricted area information
System Workflow
User submits report, Admin verifies if Public map updates.
Emergency request fi Location fetched fi Nearest camp displayed.
Technical Stack
Backend: Django
Frontend: Django Templates, HTML, CSS, JavaScript
Database: MySQL
Maps: Leaflet.js- OpenStreetMap
Project Modules- accounts- reports- camps- alerts- dashboard- core
Main Features: Incident reporting, Anonymous reports, GPS location capture, Evidence uploads, Public incident map, Missing person alerts, Admin verification panel
Security Features- CAPTCHA- Rate limiting- Duplicate detection- Identity protection
Development Roadmap
Phase 1: User authentication, Incident reports, Camp finder, Admin panel
Phase 2:- Alerts- Analytics dashboard
Phase 3: Mobile app
- Government integration
Mission
Empowering border communities through technology, safety, and accessible reporting
